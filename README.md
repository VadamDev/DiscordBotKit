# [DiscordBotKit](https://github.com/VadamDev/DiscordBotKit) - [![](https://jitpack.io/v/VadamDev/DiscordBotKit.svg)](https://jitpack.io/#VadamDev/DiscordBotKit)
A JDA based open source framework/library for discord bot creation, featuring configuration & command handling, menu management & more!

## Overview
- Bot abstraction using the JDABot class
- Annotation based configuration library
- Extended discord command API
- Console command management
- Component & Menu library
